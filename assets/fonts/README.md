# Bundled fonts for profile / combat skill cards

`DejaVuSans.ttf` and `DejaVuSans-Bold.ttf` are shipped so Railway (Linux) renders card text at full size.

Without these files, Pillow falls back to a tiny bitmap font and card text becomes unreadable.
